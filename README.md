# Nigerian-Exchange-Rate-Analysis-1950-2025-
A Power BI dashboard analyzing 75 years of Nigeria's Naira to USD exchange rate, covering depreciation trends, economic periods, and exchange rate regimes from 1950-2025.

![Dashboard Screenshot](Nigerian_Exchange_Rate_Analysis_1950-2015.png.png)

##DATA & ANALYSIS
The full data analysis is available in the Excel file included in this repository.

##Dashboard Overview
- **Highest Exchange Rate** - ₦1,554.62
- **Lowest Exchange Rate:** ₦0.55
- **Cumulative Depreciation:** 218,860.56%
- **Filters:** Depreciation Level, Economic Period, Exchange Rate Regime

## Key Insights
- The naira maintained stability from 1950 to the mid-1980s
- Rapid depreciation began after the structural adjustment period
- The FX Crisis and recent economic periods show the steepest decline
- 55.26% of years recorded moderate depreciation

## Tools Used
- Power BI Desktop (dashboard and visuals)
- Microsoft Excel (data cleaning and preparation)
- Data source: Federal Reserve Bank of St. Louis (FRED)
